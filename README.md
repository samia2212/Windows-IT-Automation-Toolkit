# 🛠️ حقيبة أدوات الأتمتة والتدقيق الأمني لإدارة الأنظمة (Enterprise IT & Security Automation Toolkit)

مجموعة من الأدوات والسكربتات المؤتمتة المطورة باستخدام **PowerShell** لمساعدة مهندسي الدعم الفني ومسؤولي الأمن السيبراني في مراقبة أداء الأجهزة، فحص الشبكات، وتدقيق الامتثال الأمني داخل المنشآت دون الحاجة لتثبيت برامج خارجية.

---

## 🚀 محتويات الحقيبة (Toolkit Projects)

### 1. فحص أداء وصحة الجهاز المحلي (Local Device Health Checker)
* **المسار:** `Device-Health-Scanner/Device_Health_Check.ps1`
* **الوصف:** أداة فحص فوري ومحلي لخصائص الجهاز، استهلاك المعالج والذاكرة، ومساحة الأقراص التخزينية، مع تصدير تقرير HTML تفاعلي منسق بال ألوان.

### 2. نظام التدقيق الموسع لشبكات الشركات (Enterprise IT Network Auditor - V2)
* **المسار:** `Enterprise-Network-Scanner/Network_Health_Auditor.ps1`
* **الوصف:** نسخة متقدمة مصممة لإدارة الشبكات الكبيرة. يدعم الفحص المتعدد للأجهزة عن بُعد (PowerShell Remoting)، حصر التحديثات الأمنية المعلقة، مراقبة أعطال الطابعات بالمنظمة، ورصد أعلى 10 عمليات مستهلكة للأداء مع استخراج الأرقام التسلسلية وموديلات الأجهزة (Asset Management).

### 3. مدقق الامتثال الأمني لأجهزة الموظفين (Local IT Security & Compliance Auditor - V2)
* **المسار:** `Local-Security-Auditor/IT_Security_Auditor_V2.ps1`
* **الوصف:** أداة تدقيق أمني متقدمة تفحص الثغرات وسوء الإعدادات الأمنية الحقيقية الشائعة في بيئات العمل (مثل بروتوكول SMBv1 المسبب لهجمات الفدية، منافذ الـ RDP المكشوفة، حظر وسائط التخزين الخارجية USB لحماية البيانات من التسريب، وسياسات قفل الشاشة). تقوم الأداة بحساب نسبة الامتثال الأمني الإجمالية للجهاز كنسبة مئوية وتوليد تقرير تفاعلي.

### 4. مدقق ومصحي أخطاء الطابعات الذكي (Printer Troubleshooter & Diagnostics)
* **المسار:** `Printer-Troubleshooter/Printer_Diagnostics.ps1`
* **الوصف:** أداة تشخيص متقدمة تقوم بفحص كافة الطابعات المتصلة (USB أو Network)، وتحليل الأخطاء العميقة (انحشار الورق، نفاد الحبر، وضع Offline، توقف خدمة Spooler)، وتوليد تقرير تفاعلي يقدم خطوات الحل الفنية والميدانية لعلاج التذكرة فوراً.
